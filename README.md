# API Test Automation

REST API test otomasyon projesi. WireMock ile mock API'lar, RestAssured ile test senaryoları.

## Geliştirici

**Burak Doğan** - [@burakdogan27](https://github.com/burakdogan27)

## Kurulum

```bash
git clone https://github.com/burakdogan27/api-test-automation.git
cd api-test-automation
mvn clean install -DskipTests
```

## Test Çalıştırma

```bash
mvn clean test
```

## Teknolojiler

- Java 17
- JUnit 5
- RestAssured
- WireMock
- Allure
- Maven
